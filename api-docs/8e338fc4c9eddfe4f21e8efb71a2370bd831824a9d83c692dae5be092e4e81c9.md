所有模块都提供这些对象。 
以下变量虽然看起来是全局的，但其实并不是。 
它们仅存在于模块范围内，请参阅[模块系统文档][module system documentation]：

- [`__dirname`]
- [`__filename`]
- [`exports`]
- [`module`]
- [`require()`]

此处列出的对象特定于 Node.js。 
有许多[内置对象][built-in objects]是 JavaScript 语言本身的一部分，它们也是全局可访问的。
