* `buffer` {Buffer|Uint8Array} 要拷贝数据的 `Buffer` 或 `Uint8Array`。

拷贝 `buffer` 的数据到新建的 `Buffer`。

```js
const buf1 = Buffer.from('buffer');
const buf2 = Buffer.from(buf1);

buf1[0] = 0x61;

console.log(buf1.toString());
// 打印: auffer
console.log(buf2.toString());
// 打印: buffer
```
